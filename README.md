TMDB Client APP
===========

a simple iOS app using Swift 2.3 that displays movies from TMDb. <a href ="http://goo.gl/eaohSg">Faris Abu Saleem</a>.

</span></a>Features
</h2>

<ul class="task-list">
<li>Written in a clean architecture</li>
<li>Code resuable</li>
